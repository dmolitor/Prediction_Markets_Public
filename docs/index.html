About This Data

This dataset contains processed data from Kalshi prediction markets, providing daily estimates of market-implied probability distributions for key macroeconomic variables including Federal Funds Rates, CPI (both headline and core), GDP growth, and unemployment rates.

The data is derived from trade-level and orderbook-level observations from Kalshi's prediction markets and is updated daily through an automated pipeline. Researchers can use this data to study how market expectations evolve over time and how they respond to economic news and policy announcements.

Data Format: All files are provided as CSV (comma-separated values) files with headers. The distribution files use long-form format where each row represents a probability mass at a specific outcome. The moments files contain summary statistics with one row per date.

Update Frequency: This dataset is updated daily at approximately 5:30 PM EST each Friday, containing data through the week.

Data collected and processed by Anthony Diercks, Jared Katz, and Jonathan Wright
